# 📚 Course Material RAG Assistant

A lightweight RAG-based question answering system for course materials.

## 🚀 Features
- Ask questions based on uploaded PDFs
- Retrieval-Augmented Generation (RAG)
- Source-grounded answers
- Streamlit interactive UI

## 🛠 Tech Stack
- Python
- LangChain
- FAISS
- Streamlit
- SiliconFlow API
